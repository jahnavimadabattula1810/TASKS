# MultiTask Management System - Frontend

This is the frontend application for the MultiTask Management System built with React, TypeScript,
and Tailwind CSS.

## Setup Instructions

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

The application will run on http://localhost:3000

## Features

- **Notes Management**: Login with email and manage notes with create, edit, and delete
  functionality
- **Support Issues**: Report and manage customer support issues with filtering and pagination
- **Resident Management**: Manage residents with full CRUD operations, filtering, and pagination

## Backend API

Make sure the Spring Boot backend is running on http://localhost:8080
